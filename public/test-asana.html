How to get a token:

Option 1: Personal Access Token (Quick & Easy)
1. Go to https://app.asana.com/0/my-apps
2. Click "Create new token"
3. Give it a name and copy the token
4. Paste it above

Option 2: OAuth Token
1. Use the OAuth Helper to authenticate
2. Copy the access token
3. Paste it above

Note: If OAuth gives you "invalid_request" error, you need to enable your app in the Asana Developer Console under "Manage Distribution"
